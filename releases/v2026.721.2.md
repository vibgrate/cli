# Vibgrate CLI 2026.721.2

_Released 2026-07-21_

This release of the Vibgrate CLI includes an important update to the vg update command, enhancing the management of the vg serve process. Users can now ensure that their environment is running the latest version without stale processes interfering.

## What changed

### New

- vg update now retires any running vg serve process executing an older version.

### Changed

- Users can opt out of this behavior with vg update --no-reap.
