# Vibgrate CLI 2026.721.1

_Released 2026-07-21_

This release of the vg command-line scanner includes improvements to symbol searching, ensuring more accurate results. Additionally, a new benchmark has been published to verify search resolution paths across multiple languages.

## What changed

### New

- A new locate-quality benchmark (bench:locate) and CI gate verify every search_symbols resolution path at 100% across TypeScript, C#, and Python fixtures.

### Improved

- --search_symbols resolves more query shapes without grep, including quoted single names, route and path strings, file paths, and C# enums and records.

### Fixed

- --search_symbols no longer returns unrelated symbols for path-like queries, providing cleaner no-match responses.
