# Vibgrate CLI 2026.709.1

_Released 2026-07-09_

This release of the CLI ships 1 new change, 1 improved change, 1 changed change and 1 performance change.

## What changed

### New

- `vg serve --share-stats` is a new opt-in that periodically shares a counts-only usage summary with Vibgrate to help improve the local AI Context server.

### Improved

- `vg savings` now shows how your assistant reaches the code map — split by MCP tools vs the `vg` CLI, and by which AI is calling.

### Changed

- The CLI now requires Node.js 22 or newer, matching the current active LTS baseline.

### Performance

- The local MCP server (`vg serve`) now reuses the code map's in-memory index across tool calls instead of rebuilding it every time, so navigation stays fast on large repositories — on a 100k-symbol map this removes roughly 200ms of rebuild work from every query after the first, an
