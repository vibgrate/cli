# Vibgrate CLI 2026.717.1

_Released 2026-07-17_

This release of the vg command-line scanner includes several important fixes and new features aimed at improving usability and integration with AI assistants. Notably, the handling of upgrade plans has been refined, and new options have been added for installation and semantic search capabilities.

## What changed

### New

- vg install gained --detect to find and install AI assistants based on the repository footprint, home config, or CLI on PATH.
- vg lsp now supports --no-graph and --no-semantic options to disable local code mapping and semantic search.
- vg can load the optional semantic-search backend from a specified directory via VIBGRATE_EMBEDDER_PATH.
- Grok CLI has been added to the list of supported AI assistants.

### Fixed

- vg fix no longer lists duplicate upgrade plans, collapsing identical upgrade sets into the lowest-risk plan.
- vg update now detects a globally installed CLI and updates it in place, avoiding unnecessary additions to project dependencies.
