# Vibgrate CLI 2026.720.1

_Released 2026-07-20_

This release of the Vibgrate CLI introduces a live status display for the vg serve command, providing real-time insights into connected AI clients and performance metrics. Additionally, vg savings now includes average response times per command.

## What changed

### New

- vg serve now shows a live status display while it runs, including uptime, connected AI clients, calls and response times per tool, and estimated context-token savings.
- vg savings reports average response time per command.
