# Vibgrate CLI 2026.704.2

_Released 2026-07-04_

This release of the vg CLI introduces several improvements to the vg serve tool and the vg install command, enhancing efficiency and reducing resource usage without altering existing capabilities.

## What changed

### Changed

- vg serve tool descriptions are now about 28% leaner in context token usage per assistant step.
- vg serve gains a search_symbols feature for fast ranked symbol and text lookup.
- vg serve now provides concise-by-default responses, reducing context costs for graph answers.
- vg install writes a deferred-loading navigation config, reducing tool overhead by about 70%.
